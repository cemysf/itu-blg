Problem 4.56: First draw the equvalent circuit using Fig. 4.26b and 4.28c. re = 10,8 Ω, gm = 92 mA/V and rπ = 1k9 were found already.

In the small signal part, vbe/vi = rπ/(RBB+rπ) - looks obviously like a voltage divider;

ib = vbe/rπ = vi/(RBB+rπ) and vo = - β*ib*RC = - β*RC*vi/(RBB+rπ)

Therefore vo/vi = - βRC/(RBB+rπ) ≈ -3.

 

Problem *4.64: Assuming β is very large, we can neglect base current, which means, VB = 7,5V, and thus VE = 6,8 V. As a result of this emitter current is found as 1 mA! Thus collector current is also very close to 1 mA. Thus re = 25Ω.

For AC analysis replace the transistor with the equivalent circuit given in Fig. 4.27b.

We can see now, that vb = vi, and therefore vo1/vi = RE/(re+RE) = 0,996.

vo2 = - α ieRC ; because base current can be neglected, ie = vi/(re+RE)

vo2 = - α ieRC = - α RC vi/(re+RE) = -vi α RC/(re+RE),

and therefore vo2/vi = - α RC/(re+RE). Assuming α ≈ 1, vo2/vi = -0,63.
