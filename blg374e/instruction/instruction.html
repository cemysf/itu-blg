How to Swap Faces in GIMP

GIMP is a free photo editing program which is used for specific tasks like photo retouching, image composition and image editing. This instruction explains swapping two areas in a photo with using basic tools.

1. Open the file in the GIMP

2. Select the first persons face with "Free Selection Tool". After that, copy and paste with Ctrl+C, then select "New Layer" from "Layer" menu

3. Apply the same procedure for other persons face. Notice that there are two additional layers for each persons face. Move the first layer to second person and second layer to first person.

4. Next, rotate and resize each layer with "Rotate Tool" from the toolbox in the left.

5. After that, "Merge Down" each layers from the layer box in the right, or from the "Layer" menu.

6. In the next step, correct the borders of each layer using "Healing Tool" from the toolbox or the "Tools" menu.
